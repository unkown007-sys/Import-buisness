How the total is calculated
Assessable value = item price + shipping
Customs duty = 10% of assessable value (flat personal-use import rate)
IGST = 18% of (assessable value + duty)
Total landed cost = assessable value + duty + IGST

This is the simplified personal-import formula — real shipments can vary by declared value, courier vs. postal route, and product category. Good enough to compare options; not a substitute for checking the exact rule at the time of import.
